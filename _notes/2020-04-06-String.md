## C
### scanf
- `scanf("%s", p)`and`scanf("%c", &c)`连续读取，首先`scanf("%s", p)`读取连续字符，直到遇到一个空格字符（空格字符可以是空白、换行和制表符），但是空格字符仍然留在缓冲区，这时可能会导致读不到想要的字符。
    ```c
    char tmp[10], c;
	scanf("%s", tmp);
	printf("%s\n", tmp);
	while(scanf("%c", &c) != EOF)
		printf("%c\n", c);
    ```
    - 输入
    ```markdown
    zxc abc
    ```
    - 输出
    ```markdown
    zxc

    a
    b
    c
    ```

- `scanf(“%[^\n]”,str)`读取字符直到遇到`\n`，`[^xx]`xx可以由编程者自己定义结束符，例如`[^ab]`则遇到`a`or`b`则停止读取，并且刚遇到的字符会留在缓冲区
    ```c
    char tmp[10], c;
        scanf("%[^\abc]", tmp);
        printf("%s\n", tmp);
        while(scanf("%c", &c) != EOF)
            printf("%c\n", c);
    ```
    - 输入
    ```markdown
    zxcabc
    ```
    - 输出
    ```markdown
    zx
    c
    a
    b
    c
    ```
